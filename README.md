# Virtual Mouse Using Hand Gestures

A computer vision project that allows users to control the mouse cursor using hand gestures through a webcam.

## Technologies Used
- Python
- OpenCV
- MediaPipe
- AutoPy
- NumPy

## Features
- Real-time Hand Tracking
- Cursor Movement
- Left Click
- Right Click
- Gesture Recognition

## Project Objective
To replace a traditional mouse with hand gesture-based control using computer vision techniques.
